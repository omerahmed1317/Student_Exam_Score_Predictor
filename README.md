# 🎓 Student Exam Score Predictor

A Machine Learning project that predicts student exam scores based on 
study habits and academic factors using **Linear Regression**.

---

## 📌 Project Overview

Most students wonder — *"How much will my exam score improve if I study more?"*  
This project answers that question using real data and machine learning!

**Input Features:**
- 📚 Hours Studied per day
- 🏫 Attendance percentage
- 📝 Previous Scores
- 👨‍🏫 Tutoring Sessions

**Output:** Predicted Exam Score (0–100)

---

## 📊 Model Performance

| Metric | Score |
|--------|-------|
| R² Score | 0.80+ |
| Mean Squared Error | ~3.5 |
| Training Samples | 5285 |
| Testing Samples | 1322 |

---

## 🔍 Key Findings

- **Attendance** has the strongest impact on exam scores (correlation: 0.58)
- **Hours Studied** is the second strongest predictor (correlation: 0.45)
- **Previous Scores** and **Tutoring Sessions** also significantly improve predictions
- Sleep hours alone showed minimal direct impact
